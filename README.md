# Sistema Bancário em Python 🏦

Projeto desenvolvido inicialmente como atividade acadêmica para praticar conceitos de Programação Orientada a Objetos (POO) em Python.

Atualmente o projeto também está sendo utilizado para aprofundar estudos sobre organização de código e tratamento de exceções.

## Funcionalidades
- criação de contas bancárias
- realização de saques
- validação de saldo
- tratamento de erros com exceções

## Conceitos utilizados
- classes e objetos
- herança
- encapsulamento
- exceções customizadas
- try/except

## Tecnologias
- Python
